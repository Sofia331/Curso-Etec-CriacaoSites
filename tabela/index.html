Como utilizar as Tabelas em HTML

As tabelas são listas de dados em duas dimensões e são compostas por linhas e colunas. Portanto, são muito utilizadas para apresentar dados de uma forma organizada

Exemplo:

Nome Valor
Água R$0,25
Suco R$0,50
Total R$0,75
